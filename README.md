# MiniReto-2
Simulacion de sistema de varios robots con el fin de completar una tarea definida
Los robots usados son los siguientes:

    - ANYmal
    - husky A200
    - PuzzleBot

## Uso

### Dependencias 

Las dependencias estas listadas en requirements.txt
Para instalarlas:

    pip install -r requirements.txt


### Ejecucion
El script sim.py ejecutara todas las simulaciones y guardara las graficas en la carpeta Results

    python3 scripts/sim.py
